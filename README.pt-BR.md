# Emmanuel Lemos — Portfólio

Projeto de portfólio pessoal desenvolvido para apresentar minha transição para o desenvolvimento frontend, unindo experiência em design com aprendizado prático em tecnologias web modernas.

O foco do projeto está em estrutura, organização e evolução contínua, sem exageros técnicos.

--------------------------------------------------

## Sobre o Projeto

Este portfólio reúne projetos, skills e experiências profissionais, com foco em desenvolvimento frontend, estrutura de interfaces e comportamento interativo.

Cada funcionalidade foi implementada com o objetivo de aprendizado prático e entendimento real do funcionamento das tecnologias utilizadas.

--------------------------------------------------

## Stack Utilizada

- HTML5
- CSS3
- Tailwind CSS
- JavaScript (Vanilla)
- Vite
- PostCSS (processamento de CSS em build)

--------------------------------------------------

## O Que Este Projeto Demonstra

- Boa base em HTML semântico e estrutura de layout
- Interfaces responsivas com Tailwind CSS
- Organização de CSS por componentes
- Lógica JavaScript aplicada a comportamento de UI
- Alternância de tema Dark / Light
- Internacionalização (EN / PT-BR) com fallback
- Conteúdo configurável via arquivos externos
- Animações e carrosséis customizados
- Preocupação com legibilidade, organização e manutenção do código

Diversas decisões técnicas foram feitas como parte do processo de aprendizado, priorizando entendimento em vez de soluções prontas.

## Como Rodar Localmente

Instale as dependências:

npm install

Execute o projeto:

npm run dev

--------------------------------------------------

## Abordagem de Configuração

O projeto separa dados de conteúdo e configurações do código principal.

- config.demo.js — configuração pública
- i18n.demo.js — dicionário de idiomas
- .env.example — referência de ambiente

Essa abordagem facilita manutenção e reutilização do projeto.
